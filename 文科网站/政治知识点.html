政治知识点
必修1 中国特色社会主义
第一课 社会主义从空想到科学
1. 原始社会
生产力低下，生产资料公有制，平均分配，无阶级无剥削。
2. 奴隶社会
生产力提高，私有制产生，阶级对立，奴隶制国家形成。
3. 封建社会
地主占有土地，农民依附于地主，地租剥削，君主专制统治。
4. 资本主义社会
根本矛盾：生产社会化与生产资料资本主义私人占有。
经济危机：根本原因是资本主义基本矛盾，表现为生产相对过剩。
5. 科学社会主义
思想来源：空想社会主义；历史条件：工人运动兴起；理论基石：唯物史观、剩余价值学说。
标志：1848《共产党宣言》发表。
6. 十月革命
1917年，建立世界上第一个社会主义国家，实现科学社会主义从理论到现实。
第二课 新民主主义革命与社会主义建设
1. 近代国情
半殖民地半封建社会，主要矛盾：帝国主义与中华民族、封建主义与人民大众。
2. 新民主主义革命
开端：五四运动（1919）；领导阶级：无产阶级；指导思想：马克思主义；1949胜利，新中国成立。
3. 社会主义制度确立
1953—1956三大改造完成，标志社会主义制度在中国建立。
4. 社会主义建设探索
中共八大正确分析矛盾；虽有失误，但取得独立工业体系基础。
第三课 伟大的改革开放
1. 开启标志
1978十一届三中全会，停止以阶级斗争为纲，以经济建设为中心，实行改革开放。
2. 进程
农村改革先行（家庭联产承包）→ 沿海开放 → 社会主义市场经济体制确立。
3. 意义
极大解放发展生产力，提升综合国力与人民生活，是决定当代中国命运的关键一招。
第四课 中国特色社会主义进入新时代
1. 新历史方位
中国特色社会主义进入新时代。
2. 社会主要矛盾
人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾。
3. 中国梦
实现中华民族伟大复兴，本质：国家富强、民族振兴、人民幸福。
4. 四个伟大
伟大斗争、伟大工程、伟大事业、伟大梦想，起决定作用的是党的建设新的伟大工程。
必修2 经济与社会
第一课 我国基本经济制度
1. 所有制结构
公有制为主体、多种所有制经济共同发展。
公有制：国有经济、集体经济、混合所有制中的国有集体成分。
非公有制：个体、私营、外资，是社会主义市场经济重要组成部分。
2. 两个毫不动摇
毫不动摇巩固和发展公有制经济，毫不动摇鼓励、支持、引导非公有制经济。
3. 分配制度
按劳分配为主体、多种分配方式并存。
按劳分配：公有制经济中劳动为尺度。
按要素分配：劳动、资本、技术、管理、数据等。
4. 共同富裕
扩大中等收入群体，增加低收入者收入，调节过高收入，取缔非法收入。
第二课 社会主义市场经济体制
1. 市场调节
机制：价格、供求、竞争。
优点：优化资源配置；缺陷：自发性、盲目性、滞后性。
2. 政府职能
实施科学宏观调控，市场监管、社会管理、公共服务、环境保护。
3. 宏观调控目标
促进经济增长、增加就业、稳定物价、保持国际收支平衡。
手段：经济（财政、货币）、法律、行政手段。
第三课 经济高质量发展
1. 新发展理念
创新、协调、绿色、开放、共享。
2. 现代化经济体系
产业体系、市场体系、收入分配体系、城乡区域发展体系、绿色发展体系、全面开放体系。
3. 乡村振兴与区域协调
解决三农问题，推动区域协调发展。
必修3 政治与法治
第一课 中国共产党的领导
1. 性质与宗旨
中国工人阶级先锋队，中国人民和中华民族先锋队；宗旨：全心全意为人民服务。
2. 执政理念
立党为公、执政为民。
3. 指导思想
马克思列宁主义、毛泽东思想、邓小平理论、“三个代表”重要思想、科学发展观、习近平新时代中国特色社会主义思想。
4. 领导方式
政治领导、思想领导、组织领导。
5. 全面从严治党
自我革命，保持先进性纯洁性，提高执政能力。
第二课 人民当家作主
1. 国体
人民民主专政的社会主义国家，本质：人民当家作主。
2. 根本政治制度
人民代表大会制度：人民选举代表，统一行使国家权力。
3. 基本政治制度
中国共产党领导的多党合作和政治协商制度；民族区域自治制度；基层群众自治制度。
4. 全过程人民民主
最广泛、最真实、最管用的民主。
第三课 法治国家、法治政府、法治社会
1. 依法治国总目标
建设中国特色社会主义法治体系，建设社会主义法治国家。
2. 法治国家
宪法法律至上，良法善治，权利保障，权力受制约。
3. 法治政府
职能科学、权责法定、执法严明、公开公正、廉洁高效、守法诚信。
4. 法治社会
全社会尊法学法守法用法，健全社会矛盾纠纷预防化解机制。
第四课 全面依法治国
1. 十六字方针
科学立法、严格执法、公正司法、全民守法。
2. 立法
尊重规律，民主立法，依法立法。
3. 执法
规范公正文明执法，全面履行政府职能。
4. 司法
以事实为依据、以法律为准绳，司法公正，独立行使职权。
5. 守法
增强全民法治观念，建设法治文化。
必修4 哲学与文化
第一单元 哲学基本原理
1. 哲学基本问题
思维与存在的关系问题：何者第一性；有无同一性。
2. 唯物论
物质决定意识，意识对物质具有能动作用；一切从实际出发，实事求是。
规律是客观普遍的，要尊重规律，利用规律。
3. 辩证法
联系观：普遍、客观、多样，整体与部分。
发展观：发展普遍性，前进性与曲折性，量变与质变。
矛盾观：对立统一，普遍性特殊性，主次矛盾，矛盾主次方面，两点论与重点论统一。
4. 认识论
实践决定认识：来源、动力、标准、目的。
真理是客观具体有条件的，认识具有反复性无限性上升性。
5. 唯物史观
社会存在决定社会意识；生产力决定生产关系，经济基础决定上层建筑；人民群众是历史创造者。
6. 价值观
价值观具有导向作用，自觉站在最广大人民立场上，实现人生价值（劳动奉献、个人社会统一、砥砺自我）。
第二单元 文化传承与创新
1. 文化特点
多样性、民族性、时代性、继承性。
2. 传统文化
相对稳定性，鲜明民族性；取其精华、去其糟粕，批判继承、古为今用。
3. 文化交流
尊重文化多样性，求同存异、兼收并蓄，推动文明互鉴。
4. 文化创新
立足实践，继承传统、推陈出新，面向世界、博采众长，以人民为中心。
5. 中国特色社会主义文化
以马克思主义为指导，坚守中华文化立场，建设文化强国，弘扬民族精神与时代精神。
选择性必修1 当代国际政治与经济
国家与国际组织
1. 国家本质
阶级统治工具，国家性质由统治阶级性质决定。
2. 政体
人民代表大会制度（中国）；总统制、议会制、君主立宪制（西方）。
3. 国家结构形式
单一制（中国）、联邦制。
4. 联合国
宗旨：维护国际和平安全，促进国际合作发展。
安理会：唯一有权采取维和行动，五大常任理事国否决权。
5. 世界多极化
多极化趋势不可逆转，和平与发展是时代主题，国际关系基本形式：竞争、合作、冲突。
中国外交与国际贡献
1. 外交政策
独立自主和平外交政策；宗旨：维护世界和平、促进共同发展；基本目标：维护主权安全发展利益，促进世界和平发展；基本准则：和平共处五项原则。
2. 人类命运共同体
持久和平、普遍安全、共同繁荣、开放包容、清洁美丽。
3. 全球治理
共商共建共享，推动国际秩序朝公正合理方向发展。
经济全球化
1. 表现
生产全球化、贸易全球化、金融全球化。
2. 影响
机遇：资源配置、技术进步、经济发展；挑战：风险跨国传递，发展不平衡。
3. 中国态度
推动经济全球化朝着更加开放、包容、普惠、平衡、共赢方向发展。
4. 对外开放格局
陆海内外联动、东西双向互济，高水平对外开放。
